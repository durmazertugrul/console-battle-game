# Battle Console Game

A simple console-based turn-based battle game built with C#, created as a personal C# learning project.

## About

This project is a work-in-progress console application where a player character fights an enemy in turn-based combat. Built from scratch to practice core C# concepts: classes, enums, properties, and control flow.

## Features

- Randomized player and enemy stats on each run
- Four player divisions: Warrior, Mage, Archer, Tank
- Four enemy divisions: Spider, Goblin, Orc, Dragon
- Turn-based combat with critical hit chance
- Colored console output for readability

## How to Run

Requires .NET SDK installed.

```bash
git clone https://github.com/kullaniciadin/repo-adi.git
cd repo-adi
dotnet run
```
